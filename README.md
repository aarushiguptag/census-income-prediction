# census-income-prediction

Overview
This project predicts whether an individual's annual income exceeds \$50K using demographic and employment-related features from the U.S. Census Income dataset. The project demonstrates the complete machine learning workflow, from data preprocessing and exploratory data analysis to model development and evaluation.


Objectives
- Analyze the Census Income dataset
- Perform data cleaning and preprocessing
- Explore relationships between features
- Build and evaluate machine learning models for income prediction


Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

Dataset
UCI Census Income Dataset (Adult Dataset)

 Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Prediction
